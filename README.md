# 2048

Python implementation of the [2048 game](https://en.wikipedia.org/wiki/2048_(video_game)).

## How to Run

```bash
python3 main.py
```

Requires Python 3.x — no external dependencies.

## Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move tiles |
| `r` | Restart |
| `q` | Quit |
